
### PROPRIEDADES FUNDAMENTAIS DE ALINHAMENTO

Existem 9 propriedades fundamentais

**6 aplicadas em containers**

`aling-content`
`justify-content`
`place-content`

`aling-items`
`justify-items`
`place-content`

**3 aplicadas em itens**

`aling-self`
`justify-self`
`place-self`


### CONTAINERS e ITEMS
Então podemos separar em 3 grupos
   y         x        xy
`aling`, `justify`, `place`


self é para itens especificos, para ele mesmo


aling é o eixo x
justify é o eixo y
PLACE é os dois eixos xy

E cada um deeles irá observar o:
- conteúdo do elemento `content`;
- itens do elemento `items`;
- o proprio elemento `self`;
