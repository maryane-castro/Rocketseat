## PROPRIEDADES FUNDAMENTAIS
Todo grid é composto de 2 princinpais grupos:
`container: o pai`e `itens: o(s) filhos`


fr -> fraçoes
#app{
    display: grid;
    grid-template-columns: 1fr 1fr 1fr;
  }

---
### CONTAINER (PAI)
- display: grid
- grid-template:
    - grid-templates-columns;
    - grid-templates-rows;
    - grid-templates-areas
- gap
    - row-gap;
    - column-gap;

## ITENS (FILHOS)
- grid-column
    - grid-column-start;
    - grid-column-end;
- grid-row
    - grid-row-start;
    - grid-row-end;
- grid-area
